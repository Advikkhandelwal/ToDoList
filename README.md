# ToDoList

This is a full-stack ToDo List application with an Express/Mongoose backend and a React/Vite frontend.

## Prerequisites

- Node.js installed
- MongoDB URI (configured in `.env`)

## Project Structure

- `src/` - Backend source code
- `frontend/` - Frontend source code (React + Vite)
- `.env` - Environment variables for the backend

## Setup & Running

This project requires running both the backend and frontend servers.

### 1. Backend Setup

Open a terminal in the root directory:

```bash

npm install

npm run dev
```

### 2. Frontend Setup

Open a new terminal and navigate to the `frontend` directory:

```bash
cd frontend

npm install

# Start the frontend dev server
npm run dev
```

The frontend will typically run on `http://localhost:5173`.
